# winter-cloud


## app

>功能

- 小程序商城后端接口

>环境

- nodejs
- mysql
- redis

>框架

- Koa
- art-template | koa-art-template
- Sequelize

>启动

```shell
cd app
yarn install
yarn dev
http://localhost:1000/
```


## api

>功能

- 基于 hapi 的 Node.js 小程序后端开发

>环境

- nodejs
- mysql

>框架

- Hapi
- Swagger
- Sequelize

>启动

```shell
cd api
node install
node api.js
http://127.0.0.1:2000/documentation
```
