# winter-cloud


## app

>功能

- 小程序后台管理系统（进行中）
- 小程序商城后端接口

>环境

- nodejs
- mysql
- redis

>框架

- koa

>启动

```shell
cd app
yarn install
yarn dev
http://localhost:1000/
```


## api

>框架

- hapi

>启动

```shell
cd api
node install
node api.js
http://127.0.0.1:2000/documentation
```


## web

>框架

- koa
- vue

>启动

```shell
cd web
node install
node web.js
http://localhost:3000/static/index.html
```
